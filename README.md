# Machine Learning

A collection of machine learning projects and data analysis notebooks.

## Project Structure

### `/Python`
Contains various machine learning notebooks and datasets:
- **Heart.ipynb** - Heart disease prediction analysis using the heart.csv dataset
- **insurance.ipynb** - Insurance data analysis using the insurance.csv dataset
- **heart.csv** - Heart disease dataset
- **insurance.csv** - Insurance dataset

### `/Try`
Experimental scripts and data:
- **global_ai_jobs.py** - Analysis script for global AI jobs data
- **global_ai_jobs.csv** - Dataset containing global AI job listings
- **none.py** - Utility script

## Getting Started

1. Clone the repository
2. Install required dependencies:
   ```bash
   pip install jupyter pandas scikit-learn numpy matplotlib seaborn
   ```
3. Navigate to the `/Python` directory and open the notebooks with Jupyter
4. Run the analysis scripts in `/Try` as needed

## Environment

- Python 3.x
- Jupyter Notebook
- pandas, scikit-learn, numpy, matplotlib, seaborn

## Notes

- Large model files are excluded from version control (see `.gitignore`)
- Notebooks should be run in order for proper context
